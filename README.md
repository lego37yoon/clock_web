# TodoSyncClock [Web]

2025학년도 1학기 임베디드소프트웨어 : Raspberry Pi 시계 화면

## 설치 및 실행

`npm install` (또는 `pnpm install`, `yarn` 사용 가능)로 패키지 설치 후, 개발 서버를 실행하여 시작할 수 있습니다.

```bash
npm run dev
```
> [!WARNING]
> 본 소프트웨어의 배경을 정상적으로 로드하기 위해서는 bg.jpeg라는 이름의 사진이 /static 폴더 아래 위치해야 합니다.
